Course Resource · Microsoft 365 Setup & File Organization Guide
📌 Pinned resource — return here any time you have a question about signing in, saving files, or submitting your work. This page is linked in every module.
In this course you write in Microsoft Word, save your work to your school OneDrive, and turn it in through Canvas. This page walks you through the whole setup from the beginning: how to sign in, where to save your files, what to name them, and how to submit them. If you joined the course partway through the year, there is a section for you near the bottom.
①  Sign in to Microsoft 365

Step 1. Go to office.com in your browser.

Step 2. Click Sign in and use your Optima school account — the email address the school gave you. Do not use a personal Microsoft account (no @outlook.com, @hotmail.com, or @live.com).

Step 3. Enter your school password. The first time, you may be asked to set a new password and add a backup phone or email. Follow the prompts — this happens only once.

Step 4. You will land on the Microsoft 365 home screen with icons for Word, OneDrive, and more. You are ready.

If you get signed into a personal account by mistake: click your profile picture (top-right) → Sign in with a different account, and use your school address.
②  📁 Set up your OneDrive folder (your “digital binder”)

All of your ELA work lives in one folder so it is easy to find and submit:

OneDrive → Optima → ELA → ELA8

To create it (one time):

  1. Open OneDrive from the Microsoft 365 home screen.
  2. Click New → Folder, name it Optima.
  3. Open Optima, create a folder named ELA.
  4. Open ELA, create a folder named ELA8.

Every file for this course goes in ELA8. The Week-1 lesson “Digital Binder Setup” walks you through this step by step.

③  File naming rule (use it for every assignment)
LastName_FirstName_AssignmentName_ELA8
Examples:
  • Smith_Jane_QuickWrite1_ELA8
  • Smith_Jane_TimeCapsule_ELA8_Draft1
  • Smith_Jane_TimeCapsule_ELA8_Final
  • Use underscores ( _ ) between parts — no spaces, no hyphens.
  • End every file with _ELA8 so your facilitator can spot the course instantly.
  • Keep every draft. Never overwrite. Save new versions as _Draft1, _Draft2, _Final.
Why it matters: your facilitator receives files from hundreds of students. A file named “essay.docx” is hard to place; a correctly named file takes five seconds and makes grading work.
④  📤 Submit through Canvas (OneDrive tab)
  1. Open the Canvas assignment and click Submit Assignment.
  2. Choose the Microsoft OneDrive tab (look for a cloud icon if you don't see the name).
  3. Browse to Optima → ELA → ELA8 and pick your file. Check the name first.
  4. Click Submit. Confirm it appears under “Submission Details.”
Submit the file, not a link. Use the OneDrive tab to attach the file itself unless the assignment specifically says to share a link.
🎉 Joined mid-year? Start here
  1. Sign in to Microsoft 365 with your school account (Section 1). No school account yet? Email your facilitator.
  2. Create your Optima → ELA → ELA8 OneDrive folder (Section 2).
  3. Bookmark this page — you will use it all year.
  4. Email your facilitator to ask which earlier assignments you should complete in M365 format and which to skip.
💬 Where to get help

About assignments, content, or your grade: message your facilitator through the Canvas Inbox (the envelope icon). Expect a reply within one business day.

Technical problems with Microsoft 365 / OneDrive: visit support.microsoft.com. Two common fixes: sign in again in a private/incognito window; or, if OneDrive won't show in Canvas, log out of Canvas, clear cookies, and log back in. If a deadline is at risk, message your facilitator before it passes.
